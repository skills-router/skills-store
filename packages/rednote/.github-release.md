# Automated npm release

This package is published by GitHub Actions whenever `main` receives a change under `packages/rednote`.

## Required repository secrets

- `NPM_TOKEN`: npm access token with publish permission for `@skills-store/rednote`

## Release behavior

- installs dependencies with `npm ci`
- reads the latest published version from npm
- bumps the package to the next patch version
- runs `npm publish --access public`
- commits the updated `package.json` and `package-lock.json` back to `main`
