# Rednote

Plan and draft Xiaohongshu content with concise, platform-ready outputs.

## Workflow

1. Confirm the goal (traffic, conversion, brand, or personal sharing).
2. Define target audience and posting angle.
3. Draft a title, body, and call-to-action in RED style.
4. Return final copy that is directly usable.

## Script

Run the bundled script:

```bash
node ./scripts/rednote.ts
```
